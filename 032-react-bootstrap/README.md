## TASK010: webpack hot reload


#### 项目运行步骤

 npm start

#### 安装react-bootstrap包

```
$ npm install react-bootstrap --save
```

[使用手册](http://react-bootstrap.github.io/)

#### 在html中引用

```
<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/latest/css/bootstrap.min.css">
```

使用`interval`来设置轮播时间
